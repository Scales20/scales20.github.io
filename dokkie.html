Dokkie

Context voor en rol binnen het project

Dit is het eerste project dat ik heb uitgevoerd voor mijn opleiding, wat diende om bekend te worden met coderen. Ik heb dit geheel zelf gemaakt.

Omschrijving en afbeeldingen

Dokkie is een simpele website die ik heb gemaakt waar je gemaakte kosten (bijvoorbeeld van een uitje) eerlijk onder meerdere mensen kan laten verdelen.
Als iemand meer heeft betaald dan eerlijk is, laat de website zien hoeveel geld die persoon terugkrijgt; anders staat er hoeveel geld deze persoon moet betalen om het eerlijk te maken.

Waarop ben ik trots?

Ik ben blij dat ik een functionele website heb weten te maken zonder enige ervaring hiervoor te hebben gehad. Verder is de kwaliteit ervan te laag vergeleken met mijn huidige werk om er trots op te zijn.
